# Trabajo Grupal: Bootstrap vs Tailwind
**IES Mar de Cádiz - 25/26**

## Descripción
Este módulo práctico desarrolla una interfaz utilizando **Bootstrap 5.3**. Se centra en la rapidez de desarrollo mediante el uso de componentes predefinidos y un sistema de rejilla flexible.

## Tecnologías Aplicadas
- **Navbar:** Barra de navegación colapsable.
- **Grid System:** Diseño responsivo en 3 columnas.
- **Modales y Tablas:** Uso de interactividad JS y estilos de datos.
- **Personalización:** Estilos CSS embebidos para efectos de hover.

## Cómo ejecutar
1. Abrir la carpeta del proyecto en **Visual Studio Code**.
2. Hacer clic derecho en `index.html`.
3. Seleccionar **"Open with Live Server"**.
