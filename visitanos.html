Aprendiendo Bootstrap 5

Somos un grupo de estudiantes de DAW que estamos aprendiendo a utilizar Bootstrap 5 para crear páginas web modernas, responsive y bien estructuradas.

Aprender Bootstrap mediante ejemplos prácticos y aplicarlo en proyectos reales.

Visión

Dominar el diseño web responsive y crear interfaces limpias y funcionales.

Valores
  • • Aprendizaje continuo
  • • Trabajo en equipo
  • • Orden y claridad
  • • Buenas prácticas

Qué hacemos

  • Creamos páginas web con Bootstrap 5.

  • Aprendemos diseño responsive y ordenado.
